## 🧑‍🏫 프로젝트 소개

<div align="center">
<h3>LG유플러스 유레카2기 미니프로젝트 익명 경매 서비스 유유상종</h3>
- <b>불필요한 물건을 경매로 판매</b> (예: SQLD 자격증 서적) <br/>
- <b>입찰 경쟁으로 자연스러운 가격 조정</b> (수요 많으면 가격 상승, 없으면 유찰) <br/>
- <b>익명 닉네임 사용으로 경매 흥미 UP</b> (예: "응 내가 살 거야~") <br/>

**→ 판매자는 높은 가격, 구매자는 합리적 가격!**
