```

실행 구성을 선택한 다음, 명령 팔레트에서 디버그: 디버깅 시작을F5 눌러 디버깅 세션을 시작합니다.
```

- <b>불필요한 물건을 경매로 판매</b> (예: SQLD 자격증 서적) <br/>
- <b>입찰 경쟁으로 자연스러운 가격 조정</b> (수요 많으면 가격 상승, 없으면 유찰) <br/>
